1
Download the extension ZIP

2
Enable Developer Mode
Chrome/Edge: chrome://extensions → Developer Mode ON · Firefox: about:debugging → Load Temporary Add-on
3
Load the extension folder
Click "Load unpacked" and select the extracted keyvault folder
4
Sign in with your KeyVault account
Use the same username + password you created here. Master password is requested once per device.
